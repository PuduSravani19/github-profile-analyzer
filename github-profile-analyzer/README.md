
# GitHub Profile Analyzer

Search any GitHub username and instantly see their profile stats, top languages, and repositories.

## 🔗 Live Demo
[your vercel link here]

## ⚙️ Built With
- React
- Context API + Custom Hooks
- GitHub REST API (no key needed)
- Tailwind CSS
- Vite

## 🚀 Features
- Search any public GitHub profile
- View followers, repos, location
- Top languages bar chart
- Repository cards with stars and forks
- Loading and error states

## 💻 Run Locally
cd github-profile-analyzer
npm install
npm run dev
